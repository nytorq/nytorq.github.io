Treehouse Project #9 - A Responsive Web App Dashboard

Description

In project 9 of 12 you're asked to build a mock web app dashboard, deploy a charting library, simulate interactivity with form fields, and to manipulate the DOM further with JS.

Skills

  • Deploying Chart.js and creating placeholder JSON data
  • How to use and manipulate inline SVGs
  • Utilizing localStorage to remember form states
  • Simulating interactivity with inputs, single-selects, and toggles
  • Deploying JS libraries for auto-complete functionality within a type-ahead

Built With

  • HTML
  • CSS
  • Javascript
  • Chart.js
  • YourApp™
