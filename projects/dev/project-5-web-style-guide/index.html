Treehouse Project #5 - A Web Style Guide

Description

In project 5 I learned how to write SCSS, create a manifest of partials, and keep styles modularized. We did this by create a style guide of commonplace web components.

Skills

  • Mobile-first web development
  • The fundamentals of SCSS (variables, mixins, functions, etc.)
  • BEM class naming conventions and how to use SCSS to create them
  • Writing nested SCSS for media queries and pseudo-classes for better readability

Built With

  • SCSS
